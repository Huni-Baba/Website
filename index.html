Build a Number Guessing Game with Python

Introduction

Welcome to the Python Game Development Course! In this course, you will learn the basics of Python programming by building a fun and interactive number guessing game. Whether you're a beginner or looking to refresh your skills, this course is designed to help you understand Python concepts in a practical way.

By the end of this course, you will:

  • Understand Python basics like variables, conditionals, and loops.
  • Learn how to use Python libraries like random.
  • Build a complete number guessing game from scratch.
  • Gain experience debugging code using Visual Studio Code.

Let's get started!

What is Visual Studio Code

Visual Studio Code (VS Code) is a free, open-source code editor developed by Microsoft. It's lightweight yet powerful, with excellent support for Python and many other programming languages. VS Code is popular among developers because it offers: Intelligent Code Completion (IntelliSense) Built-in Debugging Extensions for additional features Git Integration (version control) Terminal Access (run commands without leaving the editor)

1. Install Python & VS Code

Step 1: Install Python

  • Download Python from python.org.
  • Check installation: Open Terminal and run python --version.

Step 2: Set Up VS Code

  • Install VS Code.
  • Install the Python extension: Search for "Python" in Extensions (Ctrl+Shift+X).
  • Create a project folder and a game.py file.

7. Complete Game Code

import random

player_name = input("Enter your name: ")
secret_number = random.randint(1, 10)
attempts = 0
max_attempts = 3

print(f"Welcome, {player_name}! You have {max_attempts} attempts.")

while attempts < max_attempts:
    guess = int(input("Guess (1-10): "))
    if guess == secret_number:

    elif guess < secret_number:
        print("Too low!")
    else:
